# 📸 演示 GIF 录制清单

## 工具：ScreenToGif（Windows 免费）

1. 下载 [ScreenToGif](https://www.screentogif.com/)（推荐便携版，免安装）
2. 打开 → 录像机 → 框选浏览器窗口 → 录制
3. 编辑：裁剪头尾无用帧 → 图像 → 调整大小（宽度 800px）→ 减少帧数（间隔 2 帧）
4. 另存为 GIF：20 fps / 256 色 / 2x 编码器
